# Task 1: HTML Structure and Basic Server Interaction

A simple Node.js and Express application showcasing form handling and server-side rendering with EJS.

## Features
- **Form Submission**: Users can input their name, email, and a message.
- **Server-Side Rendering**: Uses **EJS** to dynamically display submitted data.
- **Express Backend**: A clean and efficient server setup for routing and processing.

## Prerequisites
- [Node.js](https://nodejs.org/) installed on your system.

## Setup Instructions
1. Clone this repository or download the source files.
2. Open a terminal in the project directory.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and navigate to `http://localhost:3000`.
